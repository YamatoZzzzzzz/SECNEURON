# 符号说明

| 符号 | 含义 |
|---|---|
| \(M\) | 原始语言模型 |
| \(M^A\) | 用户属性为 \(A\) 时得到的部分授权模型 |
| \(T\) | 任务集合 |
| \(t\) | 某一个任务 |
| \(D_t\) | 任务 \(t\) 的数据集 |
| \(D'_t\) | 任务 \(t\) 的测试集 |
| \(n\) | 神经元 |
| \(I(t,n)\) | 神经元 \(n\) 在任务 \(t\) 上的平均激活重要性 |
| \(G(t,n)\) | 梯度敏感性 |
| \(S(T,t,n)\) | 任务特异神经元分数 |
| \(\lambda\) | 跨任务重叠惩罚系数 |
| \(S_t\) | 任务 \(t\) 选出的神经元集合 |
| \(U\) | 神经元互斥子集集合 |
| \(k_{gt}\) | CP-ABE 管理的上层密钥材料 |
| \(k_{\text{AES}}\) | AES-CTR 使用的对称密钥 |
| \(C^M\) | 加密后的模型参数 |
| \(C^{P//K}\) | 由 CP-ABE 加密的策略与密钥 |
| \(Acc_t\) | 任务 \(t\) 的 token-level accuracy |
| PII | Personally Identifiable Information，个人可识别信息 |
| MIA | Membership Inference Attack，成员推断攻击 |
| AUROC | 攻击分类能力指标，MIA 中越接近 0.5 越接近随机猜测 |
